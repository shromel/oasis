Read the `README.md` file within this skill, and explore the other available files.

If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Fast orientation
- **`README.md`** — the full design guide: product context, content & visual foundations, iconography, manifest.
- **`styles.css`** — the single entry point. Link it and you get every token, the webfonts, and the helper classes (`.glass`, `.heading`, `.gold-text`, `.btn-gold`, `.chip`, `.input`, `.oasis-atmosphere`).
- **`tokens/`** — colors, typography, spacing, fonts, base layer.
- **`components/`** — React primitives (Button, Chip, IconTile, GlassCard, StatTile, ProgressRing, CriterionBar, LevelPill, Stepper). Each has a `.prompt.md` with usage.
- **`ui_kits/oasis-app/`** — a working iOS app recreation to copy patterns from.
- **`assets/`** — logo + app icons.

## The one-line brief
Dark, warm and layered: frosted **glass** cards floating over a **golden-oasis atmosphere** (sun above, emerald jungle, turquoise coast, faint sand grain). **Fraunces** display serif + **Inter** text. **Gold** is the one bright accent — spent carefully — with sage-green for growth/success and terracotta for streak/danger. Calm motion, press-to-shrink, generously rounded. Sophisticated minimalism that still feels alive. Never blue/purple, never sterile.
